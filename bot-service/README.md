# poc-company-ai-bot

LINE Bot powered by OpenCode + Claude AI — generated by [Botforge](https://github.com/monthop-gmail/botforge)

## Quick Start

```bash
# 1. Set credentials
cp .env.example .env
# Edit .env with your LINE credentials, API keys, and tunnel token

# 2. Deploy
docker compose up -d --build

# 3. Done! Webhook URL:
# https://sumana.online/webhook
```

## Architecture

```
LINE app → Cloudflare Tunnel → line-bot (Bun, port 3000) → OpenCode (port 4096) → AI Model
```

**3 Docker Services:**
- **opencode** — AI server with multi-model support (Claude, GPT, Gemini, DeepSeek, Qwen)
- **line-bot** — LINE webhook handler (Bun/TypeScript)
- **cloudflared** — Cloudflare tunnel (exposes webhook to internet)

## Commands

```bash
# Deploy
docker compose up -d --build

# Logs
docker logs poc-bot-line-bot --tail 30
docker logs poc-bot-server --tail 30

# Rebuild only bot
docker compose up -d --build line-bot
```

## LINE Bot Commands

| Command | Description |
|---------|-------------|
| `/new` | Start new session |
| `/abort` | Cancel current prompt |
| `/sessions` | View session status |
| `/model` | View/switch AI model |
| `/about` | About this bot |
| `/help` | Show all commands |
